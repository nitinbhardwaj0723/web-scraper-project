# 🔍 Python Web Scraper

A simple and beginner-friendly **Python Web Scraper** that extracts useful content from websites such as:

- 🌐 Website Title
- 📰 Headings (`h1`, `h2`, `h3`)
- 📄 Paragraphs (`p` tags)

The scraped data is automatically saved into separate files for easy access and analysis.

---

## 🚀 Features

- Extracts website title
- Scrapes headings (`h1`, `h2`, `h3`)
- Scrapes paragraph text
- Saves output into:
  - `title.txt`
  - `headings.csv`
  - `paragraphs.csv`
- Beginner-friendly Python project
- Useful for learning web scraping basics

---

## 🛠️ Technologies Used

- **Python**
- **Requests**
- **BeautifulSoup4**
- **Pandas**
